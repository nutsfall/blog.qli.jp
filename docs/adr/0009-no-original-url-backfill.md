# ADR-0009: 旧記事1832件のoriginal_url補完は行わない

- ステータス: 採用
- 日付: 2026-07-10（[#16](https://github.com/nutsfall/blog.qli.jp/issues/16) クローズ）（2026-07-13 事後記録）

## コンテキスト

RSS自動インポート分（42件）は `original_url` でMedium元記事を参照できるが、それ以前に一括インポートした1832件には元URL情報がない。カノニカル参照・移行履歴として補完する価値があるか検討した（#16）。

## 決定

補完しない。slugのUUIDはMedium記事IDと無関係で機械的な対応付けができず、手作業で対応表を作るコストに見合う用途（カノニカル参照の実利）もないと判断した。

## 結果・影響

- フロントマターの `source:` / `original_url:` は新旧記事でスキーマ差が残るが、これは仕様として受け入れる
- 正本はこのリポジトリ側という位置づけ（ADR-0002）がより明確になった
